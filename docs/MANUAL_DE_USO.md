# Manual de Uso - ReserveAí

Este manual resume os fluxos mais importantes da plataforma **ReserveAí** para clientes, proprietários de restaurante e administradores.

## 1. Cliente

O usuário cliente pode:

- criar conta e autenticar-se;
- consultar restaurantes disponíveis;
- visualizar detalhes do restaurante;
- criar reservas;
- acompanhar suas reservas;
- atualizar dados do próprio perfil.

### Fluxo básico do cliente

1. Acesse a página inicial.
2. Faça cadastro ou login.
3. Navegue pela lista de restaurantes.
4. Abra um restaurante específico.
5. Escolha data, horário, mesa e quantidade de pessoas.
6. Confirme a reserva.
7. Consulte o histórico em `Minhas Reservas`.

## 2. Proprietário

O proprietário pode:

- cadastrar e gerenciar restaurantes;
- cadastrar mesas;
- acompanhar reservas do seu restaurante;
- alterar status de reservas;
- evoluir a personalização White Label do restaurante.

### Fluxo básico do proprietário

1. Faça login com um usuário do tipo proprietário.
2. Cadastre um restaurante.
3. Cadastre mesas vinculadas ao restaurante.
4. Acompanhe as reservas recebidas.
5. Atualize disponibilidade e dados operacionais.

## 3. Administrador

O administrador pode:

- gerenciar usuários;
- supervisionar restaurantes;
- monitorar reservas;
- atuar em fluxos de suporte e governança.

## 4. Observações

> Algumas telas e recursos White Label foram estruturados para evolução futura no contexto acadêmico do projeto, preservando a base arquitetural para expansão do TCC.
